    • Advanced Penetration Testing Distribution

    Plague-Security is an open-source, Debian-based Linux distribution geared towards various information security tasks, such as Penetration Testing, Security Research, Multi-Task Management and Reverse Engineering

    General Knowledge

    Plague-Sec is not just all about it's tools,because it also envokes a good,high quality and flexibility in terms of platform compatability.

    Free & Opensource

    You can take any Linux and install pentesting tools on it, but you have to set the tools up manually and configure them. Plague-Sec is optimized to reduce the amount of work, so a professional can just sit down and go.

    Customization

    With Plague-Security's compatability with different platforms and with also because it's open sources it's easy to modify. You can customize the operating system the way you want and it's a good thing!

    Documentation

    Whether you are a seasoned veteran or a novice, our documentation will have all the information you will need to know about Plague Security. Multiple tips and “recipes” are available, to help ease doubts or address any issues.

    Optimized

    Developers in Plague Security are proffesional software engineers, the code is also being reviewed daily. Although bugs and issues will not be avoided it is for certain that these uncertainties will be fixed as soon as possible.

    Community

    Plague-Security doesn't have much members in it so it doesn't have much support yet on the community as a free and open source operating system. We are associated with Neutron Nation, a group made by Waren Gonzaga.
